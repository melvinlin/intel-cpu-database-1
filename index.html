关于本站

Intel CPU 数据库

本站收集整理了 Intel 近二十年（2005-2025）发布的所有主流桌面和笔记本 CPU，旨在为硬件爱好者、DIY 玩家和普通用户提供一个直观的处理器历史数据库。

数据来源

数据基于 Intel 官方 ARK 数据库、技术文档和公开评测资料整理。包含从 Core 2 Duo 到最新 Core Ultra 系列的完整代际演进。

主要特性

  • 15 个处理器代际分类展示
  • 125+ 款代表性 CPU 型号
  • 详细规格参数对比
  • 响应式设计，支持移动设备
  • 无需后端，纯静态网站

技术栈

HTML5 + CSS3 + Vanilla JavaScript
